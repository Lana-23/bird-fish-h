# Animal Images

Place animal images in this folder and name them according to the species IDs.

## Image Naming Convention

Images should be named using the species ID from the database:

### Birds
- `golden-eagle.jpg`
- `barn-owl.jpg`
- `great-tit.jpg`
- `scarlet-macaw.jpg`
- `quetzal.jpg`
- `paradise-bird.jpg`

### Fish
- `pike.jpg`
- `carp.jpg`
- `trout.jpg`
- `sea-bream.jpg`
- `grouper.jpg`
- `clownfish.jpg`
- `blue-tang.jpg`
- `lionfish.jpg`

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 300x300px or larger
- **Quality**: High quality photos for best appearance
- **Aspect Ratio**: Square or landscape preferred

## Sources

Free image sources:
- Unsplash (https://unsplash.com)
- Pexels (https://www.pexels.com)
- Pixabay (https://pixabay.com)
- Wikimedia Commons (https://commons.wikimedia.org)

## How to Add Images

1. Download or take a photo of the animal
2. Resize to appropriate dimensions
3. Save with the correct ID name in this folder
4. The app will automatically load and display the images

## Fallback

If an image is not found, the app will display the emoji placeholder instead.
